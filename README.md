# 🛡️ POSE AI Agent - Governança Novacap

Este projeto foi desenvolvido para o **GitLab AI Hackathon 2026**.

## 🚀 O que é o POSE?
O POSE (Protocolo Operacional de Segurança e Engenharia) é um agente de IA focado em **Governança Operacional**. Ele analisa descrições de obras e intervenções urbanas para identificar riscos críticos de infraestrutura (como redes elétricas e escavações) antes da execução.

## 🛠️ Tecnologias
- **Linguagem:** Python 3.x
- **Interface:** Tkinter (GUI)
- **Plataforma:** GitLab Duo Agent Platform
